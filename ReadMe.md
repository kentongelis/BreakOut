What is this project?

This project is a reacreation of the hit arcade game breakout using 
Javascript and the canvas element on HTML.

Why did I make this project?

I made this project to give myself a better foundation and a practical 
learning expierence when it came to learning Javascript.

How to run the project.

Download the directory off github and open the index.html file. Make sure you have Live Server on VSCode installed. On the index.html file,
press the Live Server button on the bottom corner of the screen to open
the game.

Tools and Technology used.

Javascript, HTMl, canvas, typescript, npm
